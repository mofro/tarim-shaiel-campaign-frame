# TOOL EVOLUTION FRAMEWORK

**Date:** 2026-01-11  
**Status:** Core Design Document  
**Purpose:** Define how legendary tools evolve alongside heroes from crude implements to mythic artifacts

---

## EXECUTIVE SUMMARY

**Core Concept:** Tools are **tethered entities** that serve as conduits for legendary power. They evolve physically and mechanically as heroes awaken to their mythic nature.

**Key Principles:**
1. Tools are **unique to each hero** (narratively distinct, mechanically varied)
2. Tool form determined by **Archetype + Class** intersection
3. Evolution mirrors **Essence progression** (crude → masterwork → ornate → legendary)
4. Tools are **limited entities** with their own nature (Moorcock-style), not passive receivers
5. Tools provide **tether to mortal world** (death mechanics, resurrection possibilities)

---

## PART 1: TOOLS AS CONDUITS

### **The Player-Facing Explanation**

"Your tool isn't just a weapon or object—it's a **conduit** for the legendary power you once wielded. In paradise, you achieved mythic feats through your heroic nature. Now, expelled and mortal, that power is **fragmented and inaccessible**... except through your tool. The tool acts as an **antenna** that can receive echoes of what you were, amplifying your mortal actions into legendary effects."

### **Why This Works**

**Mechanically:**
- Explains Charm activation: "I channel power through my tool"
- Explains whisper system: Tool remembers what hero has forgotten
- Explains R/H/K discovery: Learning what the tool knows about shared identity
- Explains binding: Tools are keyed to specific legendary nature

**Narratively:**
- Session 0: "This thing is my chain"
- Session 3: "Wait... this is my key"
- Session 8: "This IS me—the bridge between who I was and who I am"

## PART 2: TOOL EVOLUTION STAGES

### **Stage 0: Crude (Session 0 Awakening)**

**Description:**
- Rough, barely-worked implement
- Functional but unremarkable
- Clearly not "finished" or fully realized

**Examples:**
- Warrior's crude iron sword (no pommel, leather-wrapped grip)
- Seeker's rough-carved bow (unstrung when dormant, simple wood)
- Bridge's frayed rope (knotted, worn, but unbreakable)
- Visionary's clouded crystal (opaque, dim glow)

**Mechanical Properties:**
- Standard weapon/tool stats for type
- No magical properties (yet)
- Tether effect active (cannot be permanently separated)

**Narrative Moment:**
```
You wake in filth.
Leather thong bites into wrist, binding you to [crude tool].
First thought: "This is my prison."
First whisper: "[Archetype-specific guidance]"
Realization: "What IS this thing?"
```

---

### **Stage 1: Masterwork (Essence 1, ~Session 3)**

**Description:**
- Exceptional craftsmanship, clearly superior
- Still recognizable as basic tool type
- First hints of supernatural qualities

**Examples:**
- Warrior's blade sharpens to impossible edge, never dulls
- Seeker's bow string hums with tension even when undrawn
- Bridge's rope glows faintly when binding connections
- Visionary's crystal begins to clear, shows fleeting images

**Mechanical Properties:**
- Enhanced damage/range/effect (+1 tier improvement)
- First Charm tier unlocked (Tier 1 Charms accessible)
- Celestial dice progression: 2d12 Hope advantage when using tool

**Narrative Moment:**
```
[After first major revelation/flashback]
Your [tool] grows warm.
The crude implement refines before your eyes—
edges sharpen, grain smooths, weight balances perfectly.
It whispers: "Remember..."
```

---

### **Stage 2: Ornate (Essence 2, ~Session 5-6)**

**Description:**
- Visibly supernatural, ornate detailing emerges
- Tool type may shift slightly (dagger → short sword, staff → longer staff with crystal)
- Clear indicators of legendary status

**Examples:**
- Warrior's short sword, pommel bears unknown sigil, edge gleams with inner light
- Seeker's composite bow, limbs inscribed with flowing script
- Bridge's braided cord, threads shimmer with multiple colors
- Visionary's faceted orb, swirls with imagery when consulted

**Mechanical Properties:**
- Significant enhancement (+2 tier improvement)
- Second Charm tier unlocked (Tier 2 Charms accessible)
- Celestial dice: 2d12 Hope when aligned with purpose

**Narrative Moment:**
```
[After major purpose clarity moment]
Your [tool] transforms again.
What was masterwork becomes ART—
symbols emerge along its surface,
weight increases (but somehow feels lighter),
and you KNOW its name for the first time.
```

---

### **Stage 3: Legendary (Essence 3, ~Session 8)**

**Description:**
- Fully realized mythic artifact
- Unmistakably supernatural, awe-inspiring
- Final form reveals complete identity

**Examples:**
- Warrior's longsword, blazing with celestial fire when drawn
- Seeker's recurve bow, arrows manifest from pure light
- Bridge's living rope, moves with serpentine intelligence
- Visionary's perfect sphere, contains entire cosmos within

**Mechanical Properties:**
- Maximum enhancement (+3 tier improvement)
- Third Charm tier unlocked (Tier 3 Charms accessible)
- Celestial dice: 3d12 Hope advantage always active
- Unique legendary property (TBD per tool)

**Narrative Moment:**
```
[After final awakening]
Your [tool] completes its transformation.
This is what you wielded in paradise.
This is what you WERE.
The tool whispers: "We are whole again."
```

## PART 3: ARCHETYPE-CLASS TOOL MATRIX

### **How Tool Form is Determined**

**Formula:** Archetype (mythic identity) + Class (mortal skillset) = Tool Form

**Examples:**

**WARRIOR ARCHETYPE:**
- + Warrior/Guardian Class → **Blade/Hammer/Spear** (direct combat weapon)
- + Ranger Class → **Hunting Spear/Javelin** (ranged combat weapon)
- + Sorcerer Class → **Staff/Rod** (channeling combat magic)
- + Bard Class → **War Drum/Horn** (battlefield inspiration instrument)

**BREAKER ARCHETYPE:**
- + Warrior/Guardian Class → **Warhammer/Maul** (destructive force weapon)
- + Sorcerer/Wizard Class → **Key/Seal** (symbolic breaking implement)
- + Rogue Class → **Lockpick/Wire** (subtle breaking tool)
- + Druid Class → **Thorned Vine/Root** (nature's breaking force)

**BRIDGE ARCHETYPE:**
- + Bard/Seraph Class → **Thread/Rope/Cord** (literal connection)
- + Wizard/Sorcerer Class → **Mirror/Lens** (seeing connections)
- + Rogue Class → **Mask/Veil** (bridging identities)
- + Ranger Class → **Compass** (finding paths between)

**SEEKER ARCHETYPE:**
- + Ranger Class → **Bow/Crossbow** (seeking arrows, revelation shots)
- + Wizard/Sorcerer Class → **Astrolabe/Orrery** (mapping truths)
- + Rogue Class → **Spyglass/Lens** (seeing hidden)
- + Druid Class → **Lantern/Torch** (illuminating nature's secrets)

**SACRIFICER ARCHETYPE:**
- + Seraph/Guardian Class → **Bowl/Cup** (receiving/giving vessel)
- + Ranger/Warrior Class → **Surgical Knife/Scalpel** (precision healing blade)
- + Sorcerer/Wizard Class → **Censer/Thurible** (life-energy incense)
- + Rogue Class → **Bandages/Wraps** (binding wounds, concealing sacrifice)

**VISIONARY ARCHETYPE:**
- + Wizard/Sorcerer Class → **Crystal/Orb** (scrying implement)
- + Druid Class → **Incense Burner** (vision-inducing smoke)
- + Bard Class → **Scroll/Tome** (recording prophecies)
- + Rogue Class → **Mirror/Prism** (seeing through deceptions)

## PART 4: TOOL MECHANICS

### **Evolution Stats by Stage**

**WEAPON-TYPE TOOLS:**

| Stage | Damage Bonus | Range Increase | Special Properties |
|-------|--------------|----------------|-------------------|
| Crude (Stage 0) | Standard | Standard | Tether only |
| Masterwork (Essence 1) | +1 damage die size | +1 range band | Never breaks, Tier 1 Charms |
| Ornate (Essence 2) | +2 damage die size | +2 range bands | Ignores mundane armor, Tier 2 Charms |
| Legendary (Essence 3) | +3 damage die size | +3 range bands | Legendary property, Tier 3 Charms |

**NON-WEAPON TOOLS:**

| Stage | Effect Bonus | Clarity/Power | Special Properties |
|-------|--------------|---------------|-------------------|
| Crude (Stage 0) | Standard | Limited | Tether only |
| Masterwork (Essence 1) | +1 to relevant rolls | Clear/Functional | Enhanced utility, Tier 1 Charms |
| Ornate (Essence 2) | +2 to relevant rolls | Exceptional | Multiple uses, Tier 2 Charms |
| Legendary (Essence 3) | +3 to relevant rolls | Perfect/Absolute | Reality-bending, Tier 3 Charms |

### **Tether Mechanics (Death & Resurrection)**

**Tool Nearby (Within Melee Range):**
- Hero at 0 HP doesn't die immediately
- Tool blazes with desperate light
- Mechanical: Mark Stress or lose Hope, auto-stabilize
- Narrative: "The [tool] anchors you to the mortal world"

**Tool Distant (Beyond Melee Range):**
- Standard Daggerheart death rules apply
- BUT: Tool creates resurrection opportunity
- Narrative: Party can attempt to bring hero back using tool as anchor
- Discovered through play: "Wait... you were dead?"

**Tool Separated Completely:**
- Hero feels constant pull toward tool
- Cannot access Charms while separated
- Vulnerability: Standard mortal capabilities only

## PART 5: TOOL NAMING SYSTEM

### **Three-Part Identity**

```
[Archetype Quality] + [Tool Type] + [Personal Name]
```

**Examples:**
- "Sundering Hammer of Khoda" (Breaker-Warrior)
- "The Seeking Glass, Witness of Truths" (Visionary-Rogue)
- "Pathfinder's Bow, called Swift-Truth" (Seeker-Ranger)
- "The Bridge's Thread, Heartbinder" (Bridge-Bard)
- "The Sacrificial Blade, Mercy's Edge" (Sacrificer-Warrior)

### **Naming Discovery Timeline**

**Session 0:**
- Tool type only: "You have a crude sword"

**Session 2-3 (Essence 1):**
- Function revealed: "Your sword shatters chains effortlessly"
- Archetype quality emerges: "The Sundering Blade"

**Session 5+ (Essence 2+):**
- Personal name given: "I name you Khoda" (meaning "self")
- Full title emerges: "Sundering Hammer of Khoda"

## PART 6: TOOL APPEARANCE CUSTOMIZATION

### **Co-Creation Process**

**Session 0:**
- GM describes basic crude form
- Player sees/feels but doesn't customize yet

**Session 1-2:**
- GM: "What details do you notice about your weapon as you clean it?"
- Player adds: engravings, wear patterns, color, texture

**Session 3+ (Essence 1):**
- GM: "As it evolves, what changes?"
- Player describes: new symbols, material shifts, supernatural elements

**Session 5+ (Essence 2):**
- Collaborative refinement continues
- Player has significant input on ornate features

**Session 8 (Essence 3):**
- GM + Player finalize legendary appearance together
- Should reflect journey and discovered identity

## PART 7: DESIGN NOTES

### **Tool Personalities (Deferred)**

Tools are **limited entities** with their own nature (Moorcock-style blades), but specific personality frameworks will be developed after playtesting. For now:

**Guidelines:**
- Tools have preferences (expressed through whispers)
- Tools have drives (Hunger for certain actions)
- Tools have knowledge (Know what hero has forgotten)
- Tools are NOT neutral—they push toward specific behaviors

**To be developed:**
- Archetype-specific personality templates
- R/H/K whisper pattern frameworks
- GM guidance on voicing tools consistently

### **Edge Cases**

**Class Change Mid-Campaign:**
- Tool form remains constant (reflects Session 0 choice)
- Charms adjust to maintain thematic fit
- Narrative: "My tool remembers who I was, even as I change"

**Tool Damage/Destruction:**
- Tools cannot be permanently destroyed (mythic resilience)
- Can be temporarily disabled (narrative consequences)
- Regenerates when hero rests

**Multiple Tools:**
- Each hero has ONE primary tool (tether)
- Secondary items possible but mundane
- Primary tool is irreplaceable, unique bond
