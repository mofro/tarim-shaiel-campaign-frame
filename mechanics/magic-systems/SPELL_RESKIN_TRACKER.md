# DAGGERHEART SPELL RESKIN TRACKER

**Purpose:** Track alternative names for Daggerheart Domain spells/abilities that have been adapted for Tarim-Shaiel's Silk Road setting.

**Usage:** When reviewing specific Domain cards, add entries here when a spell name feels too generic or culturally misaligned. Players use Daggerheart cards at the table, but can reference these alternate names for immersion.

**Status:** Active tracking document

---

## RESKIN GUIDELINES

**When to Create Alternates:**
- Spell names that evoke European medieval fantasy (e.g., "Troubadour's Song")
- Generic magic terminology that could be more culturally specific
- Abilities that reference creatures/concepts not in Silk Road mythology

**When NOT to Reskin:**
- Mechanically clear names (e.g., "Fireball", "Healing Touch")
- Names that already feel universal (e.g., "Counterspell", "Flight")
- Domain names themselves (Arcana, Blade, Codex, etc. are abstract enough)

**Format:**
```
### ORIGINAL NAME
**Domain:** [Domain name]
**Level:** [1-10]
**Type:** [Spell/Ability/Grimoire]
**Alternate Name:** [Silk Road-flavored name]
**Cultural Context:** [Brief note on why this fits better]
**Mechanical Notes:** [Any rule clarifications needed]
```
