## About This Directory

This directory contains all project templates organized by domain:
- **Character Creation** - Character and session templates
- **Mechanics** - Charm, domain, and mechanical templates
- **World-Building** - Location, environment, and adversary templates
- **Examples** - Sample implementations and reference files
- **Kanka Integration** - Kanka sync templates and snippets
- **Tarim Shaiel** - Campaign-specific location templates

## Template Usage

1. **Choose Template** - Browse by category
2. **Copy & Rename** - Create new instance
3. **Customize** - Fill in your content
4. **Sync** - Publish to Kanka if needed
