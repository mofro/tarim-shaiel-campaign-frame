# Shared Memory Events: Interaction Trigger Architecture

## Purpose

This document tracks critical moments from the heroes' shared legendary past that will be revealed through interaction triggers (when 2+ characters work in concert).

**Design Philosophy:** Define 8-12 *moments*, not 15+ pair-specific flashbacks. Any pairing can trigger any moment through appropriate narrative conditions.

---

## Shared Memory Event Template

```
## Event: [NAME OF MOMENT]

### What This Reveals
- **Canonical Information:** What story truth does this event establish?
- **Emotional Truth:** How does each archetype experience this moment?
- **Party Impact:** Why does this matter to group cohesion?

### Typical Trigger Conditions
- **Narrative Condition:** When might this naturally emerge?
- **Mechanical Condition:** Any skill checks or actions that could trigger it?
- **Emotional Condition:** What party dynamic makes this feel earned?

### How Different Pairings Experience It

#### [Archetype] + [Archetype]
- **Their Connection:** How did these two specifically know each other?
- **Their Question:** What do they each wonder about the other?
- **Their Memory Emphasis:** What detail does THIS pairing focus on that others might miss?

[Repeat for likely pairings; others can experience the event with different focus]

### GM Notes
- Pacing guidance
- Atmospheric notes
- How to deliver this if multiple pairings trigger it
```

## Design Notes

### What Information Must Be Revealed (Across All Events)

**CRITICAL:** These canonical truths should be distributed across the event-set so that full party play reveals them organically:

- [ ] That heroes knew each other before their legendary victory
- [ ] How they came to work together (what bound them?)
- [ ] What their original purpose/charge was
- [ ] Why they didn't recognize the charge was incomplete
- [ ] What the Wizard's true role was in their story
- [ ] How each archetype failed to see something critical
- [ ] What each archetype doubts about themselves specifically
- [ ] How manufactured doubt could most effectively paralyze each archetype
- [ ] What each hero needs from the others to restore faith in themselves

### What Information Should NEVER Be Revealed (Save for Final Arcs)

- The actual mechanism by which they were expelled
- Whether they can actually return to Hero Heaven
- What the Wizard is planning or how to stop him
- Whether any hero actively betrayed the others
- The nature of the Celestial Peak and how it works

### Pacing Considerations

- Tier 1 events should emerge naturally in Sessions 0-2
- Tier 2 events should be available by Session 5
- Tier 3 events are optional, can emerge anytime party dynamics create the moment
- Never force a shared memory—let them emerge when the narrative moment feels genuine

## GM Guidance: Recognizing the Moment

### How to Know When a Shared Memory Might Emerge

Watch for:
- Two characters collaborating on something that mirrors their legendary work
- A moment where one character does something that would have been impossible before
- A conversation where one character recognizes something in another that they share
- A situation where party unity prevents failure (they *remember* they work well together)
- Environmental echoes (location that feels familiar, even if they can't say why)

### How to Deliver a Shared Memory

1. **Recognize the moment** — Let narrative happen naturally; don't force it
2. **Gather the two characters** — "You lock eyes across [the scene]"
3. **Deliver the event** — brief sensory-first description of the moment they share
4. **Let them interpret** — Don't over-explain; let them feel what it means
5. **Return to present** — The moment passes; they're back in the crisis
6. **Optional: Discuss after** — In downtime, they might compare what they saw/felt
