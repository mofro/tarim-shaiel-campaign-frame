## HERO HEAVEN COST CONVERSION

**This reference shows ORIGINAL Exalted costs. For Hero Heaven conversions:**

### Conversion Formula

```
Exalted Mote Cost → Hero Heaven Hope:
1-3m → 1 Hope
4-6m → 2 Hope
7-10m+ → 3 Hope

Willpower additions (+1wp) → +1 Hope
Initiative costs (i) → Convert case-by-case or drop
Permanent Charms (—) → Permanent (unlock once, always active)

Exalted Essence → Hero Heaven Tier (Daggerheart Levels):
Essence 1-2 → Tier 1-2 (Levels 1-4)
Essence 3-4 → Tier 2-3 (Levels 2-7)  
Essence 5 → Tier 3-4 (Levels 5-10)
```

### Example Conversions

```
Exalted: Cost: 3m; Type: Supplemental
Hero Heaven: Cost: 1 Hope; Type: Supplemental

Exalted: Cost: 5m, 1wp; Type: Simple  
Hero Heaven: Cost: 2 Hope + 1 Hope; Type: Simple

Exalted: Cost: 10m, 1wp; Type: Reflexive
Hero Heaven: Cost: 3 Hope + 1 Hope; Type: Reflexive
```

## HERO HEAVEN ADAPTATION NOTES

### Archetype Assignments
See `CHARM_ACCESSIBILITY_MATRIX.md` for which archetypes access which Charms.

### R/H/K Integration
Each Charm will have tool whispers integrated:
- **Resist:** Tool guidance to NOT use Charm in certain ways
- **Hunger:** Tool urges restoration/use of Charm
- **Know:** Tool reveals fragmentary truths about Charm's consequences

### Tier Mapping
- **Essence 1-2** → **Hero Heaven Tier 1** (Sessions 2-3)
- **Essence 3-4** → **Hero Heaven Tier 2** (Sessions 5-6)
- **Essence 5** → **Hero Heaven Tier 3** (Session 8+)

### Design Philosophy
- Hope costs keep Charms tactical (limited daily use)
- Permanent Charms provide character identity
- High-cost Charms (4-5 Hope) are campaign-defining moments
- R/H/K whispers add narrative tension to mechanical choices
