## HERO HEAVEN COST CONVERSION

**This reference shows ORIGINAL Exalted costs. For Hero Heaven conversions:**

### **Conversion Formula:**

```
Exalted Mote Cost → Hero Heaven FP/Hope:
1-3m → 2 FP OR 1 Hope (Tier 1)
4-6m → 3 FP OR 1 Hope + 1 Will (Tier 2)
7-10m → 2-3 Hope (Tier 3, Hope-only)

Willpower additions (+1wp) → +1 Will
Initiative costs (i) → Rare, converted case-by-case or dropped
```

### **Example Conversions:**

```
Exalted: Cost: 3m; Type: Supplemental
Hero Heaven: Cost: 2 FP OR 1 Hope; Type: Supplemental

Exalted: Cost: 5m, 1wp; Type: Simple  
Hero Heaven: Cost: 3 FP OR 1 Hope + 1 Will; Type: Simple

Exalted: Cost: 10m, 1wp; Type: Reflexive
Hero Heaven: Cost: 3 Hope; Type: Reflexive
```

### **Resource Definitions:**

**FP (Fatigue Points):**
- Derived from Health stat
- Tactical resource, recovers with rest
- Used for most Charms

**Hope:**
- Strategic/narrative resource
- Gained from duality dice + resisting whispers + Tier bonuses
- Used for Tier 3 Charms or as alternative payment

**Will:**
- Derived from Will stat
- Required for powerful Tier 2+ Charms
- Recovers end of session
