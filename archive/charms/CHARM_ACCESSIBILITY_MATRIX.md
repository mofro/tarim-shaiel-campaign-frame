## CHARM COST STRUCTURE (HYBRID FP + HOPE SYSTEM)

**Hero Heaven uses a hybrid resource system for Charm activation:**

### **Three Primary Resources:**

**1. Fatigue Points (FP) - Tactical Resource**
- Derived from Health stat (typically 10-12 FP)
- Spent: Most Charms (1-3 FP per activation)
- Recovery: Short rest = half FP, long rest = full FP
- Purpose: Frequent Charm use without resource starvation

**2. Hope - Strategic/Narrative Resource**
- Gained: Duality dice (rolling with Hope), resisting whispers, mythic tier bonuses
- Spent: Tier 3 Charms, Help actions, optional payment for any Charm
- Recovery: Rarely (multi-session, narrative moments)
- Purpose: Precious resource for high-impact effects + narrative signal

**3. Will Points - Conviction Resource**
- Derived from Will stat (typically 3-5 Will)
- Spent: Tier 2+ Charms requiring conviction (1 Will in addition to FP/Hope)
- Recovery: End of session or major narrative moment
- Purpose: Strategic resource for powerful effects

### **Charm Cost Patterns:**

**Tier 1 Charms:**
```
Cost: 2 FP OR 1 Hope
Examples: Perfect Strike, Truth Sense, Empathy Reading
```

**Tier 2 Charms:**
```
Cost: 3 FP OR 1 Hope + 1 Will
Examples: Iron Defense, Systematic Unraveling, Power Tracing
```

**Tier 3 Charms:**
```
Cost: 2-3 Hope (Hope-only, no FP option)
Examples: Celestial Fury, Apocalyptic Potential, Resurrection
```

### **Cost Conversion from Exalted:**

```
Exalted Mote Cost → Hero Heaven FP/Hope:
1-3m → 2 FP OR 1 Hope
4-6m → 3 FP OR 1 Hope + 1 Will
7-10m → 2-3 Hope (Hope-only)

Willpower additions (+1wp) → +1 Will
Initiative costs → Rare, converted to Hope or dropped
```

## HOPE ECONOMY

### **Hope Gain:**
- Rolling with Hope on duality dice: +1 Hope
- Resisting tool whispers: +1 Hope (always)
- **Mythic Tier Bonuses (NEW):**
  - Tier 1: +1 Hope at start of each scene
  - Tier 2: +2 Hope at start of each scene
  - Tier 3: +3 Hope at start of each scene

### **Hope Spending:**
- Tier 3 Charms: 2-3 Hope
- Help actions: 1 Hope
- Optional: Pay Hope instead of FP for any Charm (1 Hope = 3 FP value)

## FICTIONAL REQUIREMENTS (COMPETENCY AREAS)

**Instead of hard GURPS skill prerequisites, Charms use fictional gates:**

**At character creation, players define 3-5 competency areas:**
- Examples: "Melee combat, horseback riding, intimidation"
- Examples: "Archery, stealth, wilderness survival"
- Examples: "Diplomacy, ancient lore, occult knowledge"

**Charms require fictional consistency:**
- Can't use bow Charms without archery competency
- Can't use medicine Charms without healing competency
- Can expand competencies during campaign (training, XP investment)

**GM approves Charm selection based on:**
- Does this fit character's established competencies?
- Has character narratively developed this area?
- Does this match archetype's core identity?

## TOOL WHISPER FRAMEWORK

**Every Charm includes tool whisper interaction:**

**Standard Charm (Whisper Optional):**
- Tool whispers guidance
- Following = celestial advantage dice (2d12/3d12 Hope + 1d12 Fear)
- Resisting = standard dice + gain 1 Hope
- Charm works either way

**Guidance-Dependent Charm:**
- Tool whispers specific target/approach
- Following = Charm activates + advantage dice
- Resisting = Charm doesn't activate, gain 1 Hope instead

**Knowledge Charm:**
- Tool provides information + interpretation
- Following interpretation = advantage on related rolls
- Resisting interpretation = standard rolls, own conclusions

## DESIGN NOTES

**Why Hybrid FP + Hope:**
- Prevents Hope starvation (FP enables frequent Charm use)
- Preserves Hope as strategic/narrative resource
- GURPS stats matter (Health/Will affect resource pools)
- Players have meaningful choice (spend FP or Hope?)
- Tier 3 Charms feel precious (Hope-only, no FP option)

**Why Competency Areas Instead of Hard Prerequisites:**
- Fictional coherence without GURPS complexity
- Players signal character direction
- Can expand competencies during play
- GM approves based on narrative, not stat checks

**Why Tool Whispers:**
- Creates Follow/Resist binary choice
- Integrates with celestial dice progression
- Supports R/H/K discovery through mechanical choice
- Every Charm becomes identity-defining moment
