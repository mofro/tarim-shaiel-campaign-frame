# HERO HEAVEN CHARM SYSTEM

**Last Updated:** 2025-12-31

---

## OVERVIEW

The Hero Heaven Charm system adapts Exalted 3E's proven Charm structure to work with Daggerheart's duality dice and Tarim-Shaiel's R/H/K discovery framework. Heroes recover divine abilities (Charms) as they awaken across three Tiers, using a hybrid FP + Hope resource system.

**Core Design:**
- **Hybrid Resources:** Most Charms use Fatigue (tactical) OR Hope (strategic)
- **Tier Gates:** Narrative awakening moments unlock Charm tiers, not session numbers
- **Tool Whispers:** Every Charm creates Follow/Resist choice with celestial dice advantage
- **Competency Areas:** Fictional requirements replace hard GURPS prerequisites
- **R/H/K Integration:** Charms drive identity discovery through mechanical choice

## RESOURCE SYSTEM

### **Fatigue Points (FP) - Tactical Resource**

**What:** Derived from Health stat (typically 10-12 FP)

**Recovery:**
- Short rest: Recover half FP
- Long rest: Recover full FP
- Between scenes: GM discretion

**Used For:**
- Tier 1 Charms: 2 FP
- Tier 2 Charms: 3 FP
- Most frequent Charm activations

**Purpose:** Enables frequent Charm use without Hope starvation

---

### **Hope - Strategic/Narrative Resource**

**Gained:**
- Rolling with Hope on duality dice: +1 Hope
- Resisting tool whispers: +1 Hope (always)
- **Mythic Tier Bonuses:**
  - Tier 1: +1 Hope at start of each scene
  - Tier 2: +2 Hope at start of each scene
  - Tier 3: +3 Hope at start of each scene

**Used For:**
- Tier 3 Charms: 2-3 Hope (Hope-only, no FP option)
- Help actions: 1 Hope (Daggerheart)
- Alternative payment: Can pay Hope instead of FP for any Charm

**Recovery:** Rarely (multi-session, narrative moments)

**Purpose:** Precious resource for high-impact effects + narrative signal of alignment

---

### **Will Points - Conviction Resource**

**What:** Derived from Will stat (typically 3-5 Will)

**Used For:**
- Tier 2+ Charms requiring conviction: +1 Will (in addition to FP/Hope)
- Powerful effects that demand mental fortitude

**Recovery:** End of session or major narrative moment

**Purpose:** Strategic resource for high-tier Charms

---

### **Initiative - Combat Momentum (Rare)**

**What:** Derived from Basic Speed (typically 6-7)

**Used For:** Rare combat Charms trading turn order for power

**Recovery:** Per combat round

**Purpose:** Tactical tradeoff in high-stakes combat (rarely used)

## CHARM COST PATTERNS

### **Tier 1 Charms:**
```
Cost: 2 FP OR 1 Hope
Examples:
- Warrior's Perfect Strike (+1 success to attack)
- Seeker's Truth Sense (detect lies)
- Bridge's Empathy (read emotions)
```

### **Tier 2 Charms:**
```
Cost: 3 FP OR 1 Hope + 1 Will
Examples:
- Warrior's Iron Defense (Defense vs. environmental hazards)
- Breaker's Systematic Unraveling (disable organizations)
- Seeker's Power Tracing (see political connections)
```

### **Tier 3 Charms:**
```
Cost: 2-3 Hope (Hope-only)
Examples:
- Warrior's Celestial Fury (multiple attacks without penalty)
- Breaker's Apocalyptic Potential (destroy entire area)
- Sacrificer's Resurrection (bring dead back to life)
```

## FICTIONAL REQUIREMENTS (COMPETENCY AREAS)

**Instead of hard GURPS prerequisites, Charms use competency areas:**

### **At Character Creation:**

Players define 3-5 competency areas:
```
Examples:
- "Melee combat, horseback riding, intimidation"
- "Archery, stealth, wilderness survival"
- "Diplomacy, ancient lore, occult knowledge"
```

### **During Campaign:**

Players can expand competencies:
- Narrative training ("I spend downtime learning riding")
- XP investment ("I invest 10 XP in Riding skill")
- GM approval based on fiction

### **Charm Selection:**

GM approves Charms that fit:
- Does this match established competencies?
- Has character narratively developed this area?
- Does this fit archetype's core identity?

**Purpose:** Fictional coherence without GURPS complexity tracking

## TOOL WHISPER FRAMEWORK

### **Standard Charm (Whisper Optional):**
- Tool whispers guidance
- **Following:** Celestial advantage (2d12/3d12 Hope + 1d12 Fear) on related rolls
- **Resisting:** Standard dice (1d12 Hope + 1d12 Fear) + gain 1 Hope
- Charm works either way

### **Guidance-Dependent Charm:**
- Tool whispers specific target/approach
- **Following:** Charm activates + advantage dice
- **Resisting:** Charm doesn't activate, gain 1 Hope instead

### **Knowledge Charm:**
- Tool provides information + interpretation
- **Following interpretation:** Advantage on related rolls
- **Resisting interpretation:** Standard rolls, own conclusions

## TIER PROGRESSION

**Tiers unlock narratively, not by session number:**

### **Tier 0 (Unaware):**
- No Charms (divine memories dormant)
- GURPS skills define competency
- Duality dice for conflict resolution

### **Tier 1 (First Awakening):**
- First flashback (heroic deed remembered)
- First Charm recovered
- Celestial advantage when using tool (2d12 Hope)
- +1 Hope per scene

### **Tier 2 (Purpose Clarity):**
- Purpose flashback (WHY they were heroic)
- Tier 2 Charms unlock
- Celestial advantage when aligned (3d12 Hope)
- +2 Hope per scene
- R/H/K discovery moment

### **Tier 3 (Full Awakening):**
- Complete memory flashback
- Tier 3 Charms unlock
- Celestial advantage always (3d12 Hope)
- +3 Hope per scene
- Actual R/H/K locked in

## ARCHETYPE DISTRIBUTIONS

### **Warrior** (Melee, Resistance, Athletics)
- ~50-60 primary Charms
- Focus: Direct combat, endurance, physical prowess

### **Breaker** (Melee, Presence, Bureaucracy)
- ~40-50 primary Charms
- Focus: Destruction, intimidation, system-breaking

### **Bridge** (Presence, Socialize, Bureaucracy)
- ~40-50 primary Charms
- Focus: Diplomacy, connection, understanding relationships

### **Seeker** (Investigation, Awareness, Larceny, Lore)
- ~60-70 primary Charms
- Focus: Discovery, secrets, pattern recognition

### **Sacrificer** (Medicine, Resistance, Survival)
- ~30-40 primary Charms
- Focus: Healing, endurance, self-sacrifice

### **Visionary** (Occult, Lore, Awareness)
- ~50-60 primary Charms
- Focus: Prophecy, mystical knowledge, seeing beyond

## INTEGRATION WITH OTHER SYSTEMS

### **Celestial Dice Progression:**
- Charms create Follow/Resist choices
- Following = advantage dice (2d12/3d12 Hope)
- Resisting = standard dice + Hope gain
- See: /mechanics/CELESTIAL_DICE_MECHANICS.md

### **R/H/K Discovery:**
- Aspirational R/H/K at Session 0
- Charms reveal misalignment through play
- Resisting whispers = boundary discovery
- Actual R/H/K locked at Tier 3
- See: ../narrative/HERO_IDENTITY.md

### **GURPS Character Advancement:**
- Health investment = larger FP pool
- Will investment = more powerful Charm uses
- Skill investment = expanded competency areas
- XP signals character direction for Charm selection
